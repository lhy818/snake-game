# 🐍 贪吃蛇 AI 游戏

一个带有 AI 自动驾驶功能的现代贪吃蛇游戏。

## ✨ 特性

- 🎮 **经典贪吃蛇玩法** - 控制蛇吃食物，避免撞墙或撞到自己
- 🤖 **AI 自动驾驶** - 智能 AI 可以自动玩游戏
- 🎨 **炫酷视觉效果** - 霓虹风格 UI，粒子背景动画
- 🔊 **音效系统** - Web Audio API 实现的游戏音效
- 📱 **响应式设计** - 支持桌面和移动设备
- ⚡ **多种速度** - 慢速、中速、快速、极速模式

## 🚀 快速开始

### 在线运行

```bash
npx serve
```

然后在浏览器中打开 http://localhost:3000

### 直接打开

直接双击 `index.html` 文件即可在浏览器中运行。

## 🤖 AI 自动驾驶

点击 **🤖 AI** 按钮启用 AI 自动驾驶模式。

### AI 算法特性

- **尾巴追踪** - 始终保持能到达尾巴的路径，确保有逃生路线
- **空间感知** - 使用 Flood Fill 算法计算可达空间
- **自适应策略** - 根据蛇的长度自动调整行为：
  - 📈 **前期**（填充率 < 30%）：激进追食物
  - ⚖️ **中期**（30-50%）：平衡模式
  - 🛡️ **后期**（> 50%）：保守模式，优先保证空间
- **综合评分系统**：
  - 可达空间越大越好
  - 尾巴可达加分
  - 食物距离影响得分

## 📊 评分系统

- 每吃一个食物 +10 分
- 最高分自动保存到本地存储

## 🛠️ 技术栈

- **HTML5 Canvas** - 游戏渲染
- **Vanilla JavaScript** - 游戏逻辑
- **CSS3** - 动画和样式
- **Web Audio API** - 音效
- **LocalStorage** - 最高分保存
